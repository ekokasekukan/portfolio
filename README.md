# portfolio
Здесь представлена программа coffee2.cpp - проект по ООП. Проект еще находится в процессе доработки. 
Суть в том, что пользователь покупает кофейню, нанимает бариста и может посмотреть на развитие кофейни взависимости от изначальных данных:
потока гостей, зарплаты и производиельности бариста и т.п. Для этого есть два режима: быстрый(прожить неделю и получить итоговый прогресс) 
или подробный(прожить день, увидеть заказы гостей и в конце дня должны быть обработаны рандомные отзывы). Если пользователь накопит деньги, то может купить новую кофейню. Если же обанкротиться, то игра завершается.
